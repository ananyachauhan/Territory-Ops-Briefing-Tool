# Territory Ops Briefing Tool

Tesla-inspired React app for generating leadership and technician territory briefings via Groq (Llama 3 70B).

## Setup

```bash
npm install
cp .env.example .env
# Add your GROQ_API_KEY to .env
```

## Development

Runs the Vite frontend (port 5173) and Express API (port 3001):

```bash
npm run dev
```

## Build

```bash
npm run build
```
